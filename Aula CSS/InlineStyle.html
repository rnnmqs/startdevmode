Internal Style Sheet

Essa linha está formatada pelo Internal CSS declarado dentro do escopo HEAD utilizando a tag < style> e abaixo as tags que vao ser personalizadas

< style type ="text/css">
p{color:darkblue;font-family:Tahoma;}
h1{color:grey;font-family: Tahoma;}
< /style>

Conceito de classes e ID's

Para utilizar classes inicia com o .+nome da classe{...} e pode ser utilizado diversas vezes dentro do codido.

Os ID's são unicos dentro da pagina, ou seja, só podem ser utilizados uma unica vez, ele inicia com #+nome do ID{...}.
