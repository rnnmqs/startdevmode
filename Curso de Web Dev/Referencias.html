Comandos de formatação de texto

A tag < p > é utilizada para fazer paragrafos

O Comando < hr > cria uma linha automaticamente.
O comando < strong > Deixa o texto em negrito.
O comando < em > coloca o texto em Italico.
O comando < u> deixa o texto sublinhado
O comando < strike> faz um risco no meio da palavra
O comando < img src> adiciona imagens.
O comando < !-- inicia o comentario e --> fecha.
O comando < img src> carrega imagens e para definir tamenho utiliza width e height

Tabelas

O comando < table> cria tabelas, atributo border="" adiciona bordas, a linha da tabela inicia com < tr> e para inserir dados utiliza o < td>, Para os titulos das tabelas podemos utilizar o < th> exemplo:
titulo 1 titulo 2 Titulo 3
1 2 3

Para agrupar colunas utiliza colspan dentro da TD, exemplo

titulo 1 titulo 2 Titulo 3
1

Para agrupar linhas itiliza o rowspan dentro da td, exemplo:
titulo 1 titulo 2 Titulo 3
1 2
